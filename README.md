# Graphics-Programming-Lab

컴퓨터 그래픽스 관련 학습 및 과제 기록입니다. OpenGL 기초부터 Vulkan을 이용한 렌더링 파이프라인 제어까지의 과정을 포함합니다.

## 구성

### 01-Academic-Courses (전공 수업)
* 3-1 Computer-Graphics (OpenGL): 그래픽스 파이프라인 기초 및 셰이더 프로그래밍.
* 3-2 Advanced-Graphics (Vulkan/OpenGL): Vulkan API 모델 렌더링 및 씬 매니저 설계.

### 02-Self-Study-Log (개인 학습)
* OpenGL-SuperBible: API 기본기 강화 및 예제 실습.
* Vulkan-API-Practice: Vulkan 메모리 할당 및 렌더 패스 최적화 학습.

## 기술 스택
* C / C++, OpenGL, Vulkan, GLSL, HLSL
