## Getting Started

1) Clone this project (or download it as a ZIP file)
2) Open `CG_OpenGL_Assignment02/Assignment02-1/OpenglViewer.sln`<br>
![image](https://github.com/user-attachments/assets/25264591-53ef-4621-9dca-e65eb49dc01d)
![image](https://github.com/user-attachments/assets/5eef6100-4d1c-4080-8c71-81811c9893ec)
![image](https://github.com/user-attachments/assets/57347965-b3d4-40ab-8a48-75dcda40d0eb)


5) Open `Main_EmptyViewer.cpp` in the `OpenglViewer` project  
   ![image](https://github.com/user-attachments/assets/a789387d-2104-4a08-b97a-ba7913a0795f)

6) In the Solution Explorer, right-click the project you want to execute, then choose 'Set as StartUp Project' to make it the default project to run.

   ![image](https://github.com/user-attachments/assets/8179841c-3e80-490a-b3fc-f75234d46971)
   ![image](https://github.com/user-attachments/assets/e6fc2d95-6d51-47fd-9ae4-f40c1619b04f)
   
✅ If the startup project is set correctly, its name will appear in **bold** in the Solution Explorer:

![Bold Project Name](https://github.com/user-attachments/assets/6256caba-4ea0-477e-a21b-e4782f5498b8)

6) Press **F5** to build and run the project
