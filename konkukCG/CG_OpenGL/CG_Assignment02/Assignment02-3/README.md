# Assignment 02-3

Assignment 02-3 focuses on applying antialiasing to the final rendered image.  
Before Version Project, each pixel is sampled at the fixed center position (0.5, 0.5),  
but instead, 64 sample values are collected from random positions within the pixel area,  
the average of these sampled values is then used to determine the ray.  
-> antialiasing!!
